## MVP recomendado

O MVP deve provar o ciclo principal:

> Cliente compra → paga → serviço é criado → acesso é enviado → fatura renova → suporte tem contexto.

Escopo do MVP:

- Multi-tenant
- Admin
- Cliente
- Produtos
- Planos
- Checkout
- Faturas
- PIX
- Cartão
- Clientes
- Serviços
- Provisionamento cPanel ou VPS
- Tickets
- E-mails automáticos
- Dashboard
- Logs
- Design system forte
