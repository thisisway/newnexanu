# Nexano

**Nexano** é uma plataforma SaaS multi-tenant para empresas venderem, cobrarem, provisionarem, automatizarem e gerenciarem serviços digitais recorrentes.

O objetivo é criar uma alternativa moderna, intuitiva e superior ao WHMCS, com foco em:

- Billing recorrente
- Loja e checkout
- Clientes e área do cliente
- Serviços digitais
- VPS, hospedagem, domínios e cloud apps
- Suporte com contexto
- Automação visual
- Provisionamento em tempo real
- Design system premium
- Experiência simples para clientes finais
